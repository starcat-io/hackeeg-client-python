HackEEG Python client library
=================================================

Release v\ |version|. (:ref:`Installation <install>`)

.. toctree::
   :maxdepth: 2
   :caption: Contents:

**HackEEG** allows you to communicate with the `HackEEG shield <Python clien://github.com/starcat-io/hackeeg-shield>`_
for the `TI ADS1299 EEG system-on-a-chip <http://www.ti.com/product/ADS1299>`_,
configure the board, and read and stream data. Sample programs are provided that can stream data to other software
via `Lab Streaming Layer <://github.com/sccn/labstreaminglayer>`_.
